Setup steps

  1. Buy Starter and download the package.
  2. Create a GitHub repository and upload the project.
  3. Create a new Render Web Service and connect that repository.
  4. Add the build command and start command below.
  5. Add the Starter variable block below.
  6. Deploy, then connect your assistant to /mcp.
