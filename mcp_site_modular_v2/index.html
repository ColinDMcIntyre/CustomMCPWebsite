Built to be practical

I built this for myself because I wanted a simple way to connect AI to real tools without paying for another monthly service. Then I cleaned it up, made it modular, and documented it so other people could use it too.

Built and supported directly by Colin McIntyre. If you have setup questions, you can contact me directly.

Choose the setup that fits you

Starter covers the core Google setup. Pro adds more tools. Custom is for private integrations or a setup built around your own workflow.
