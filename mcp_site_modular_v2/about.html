About

Built by someone who actually needed it

I made this because I wanted AI to connect to real tools without paying another middle layer every month, and without restarting from scratch every time I wanted a new server. After I got it working for myself, I kept the useful parts, cleaned up the docs, and packaged it in a way that other people could actually follow.

What changed from the earlier version

  • The site now leads with the value of the package instead of drowning people in setup language.
  • The docs separate defaults, generated secrets, and account-based values.
  • Pricing is simpler so someone can actually decide without comparing too many tiers.
  • The public package is focused on the parts that are more stable and useful.

How I think about it

This is not meant to pretend to be some giant polished platform. It is a practical scaffold for people who want something real to start from and then keep building on.

If you need more than that, that is what the custom help path is for.
