Starter plus Maps, Memory, and Alpaca

Pro uses the same basic setup as Starter, then adds the extra variables for Maps, Memory, and Alpaca. It is for people who want more tools without changing the overall deployment process.

Setup steps

  1. Buy Pro and download the package.
  2. Push the project to GitHub.
  3. Create a Render Web Service and connect the repository.
  4. Add the same build and start commands used in Starter.
  5. Add the Starter variable block first.
  6. Add the Pro block to turn on Maps, Memory, and Alpaca.
